h1. favicon.js helps get any site's favicon URL

h2. installation?

<code>npm install favicon</code>

(you must "install npm":https://github.com/isaacs/npm first)

h2. how do I use it?

<code>./favicon dtrejo.com</code>

h2. how do I use it with my code?

<code>var favicon = require('favicon');
var icon = favicon.get('dtrejo.com');</code>

h2. heritage?

favicon.js is a Javascript/node.js port of "braveicon":https://github.com/bmelton/Braveicon/tree/master by "bmelton":https://github.com/bmelton.

<quote>Fetches a site's favicon URL.  Respects link rel="" declarations
most importantly, per W3C spec, however, it favors 'shortcut icon'
over /favicon.ico, as many software packages explicitly declare
via shortcut icon, and W3C snubs /favicon.ico, generally speaking<quote> &mdash; "bmelton":https://github.com/bmelton
